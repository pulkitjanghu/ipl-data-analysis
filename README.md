# IPL Data Analysis Project

This is a mini data analysis project on IPL using Python, Pandas and Matplotlib.

## Key Insights:
- Total matches played each season
- Matches won by each team
- Toss impact on match result
- Top 10 bowler with most wicket in ipl
- Top 10 batsman with most run in ipl

- ## Tools Used:
- Python
- Pandas
- Matplotlib

## Dataset:
- matches.csv
- deliveries.csv (both from Kaggle)
